Virtual environments y Pip

Pip

Pip es el manejador de paquetes en python.


Para instalarlo en Ubuntu: 

      $ sudo easy_install pip
      $ sudo pip install --upgrade pip
Para instalarlo en MacOS: 

      $ sudo easy_install pip
      $ sudo pip install --upgrade pip
Para instalarlo en Windows: 

      $ python get-pip.py

Al igual que python a veces se corre como python o python3, igual pip o pip3.

Virtual environment

Ya que tenemos pip vamos a crear un virtual environment, estos nos sirven para que los paquetes que instalemos de pip esten separados, muchas veces vamos a tener dos paginas con librerias diferentes y no queremos que unas esten instaladas en la otra pagina. Primero en la terminal se tiene que estar en el directorio donde la aplicacion va a estar.


Para crear un nuevo enviroment: 
        $ python3 -m venv nombre-del-env
Para crear activarlo en Linux y MacOS: 
        $ source nombre-del-env/bin/acivate
Para crear activarlo en Windows: 
        $ \env\Scripts\activate.bat
