Debug

Con lo que se ha visto si la aplicacion llega a tener un error en estos momentos no sabriamos cual es, ya que si hay algun error solo muestra una pagina en blanco o incluso si hacemos algun cambio probablemente la pagina no se va actualizar. Para lograr esto hay que activar el modo debug poniendo el modo debug en true cuando la corremos (este cambio se hace en el archivo app.py): 

    
      if __name__ == "__main__":
          app.run(debug=True)

De esta manera cuando haya un error nos debera aparecer una pagina como esta:

Por ejemplo en la imagen de arriba nos dice que el archivo debu.html no se encontro, si seguimos scroleando para abajo podemos encontrar la linea exacta en la cual esta el error.
